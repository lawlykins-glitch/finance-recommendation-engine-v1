# Finance Recommendation Engine v1

## Overview

This project is an explainable machine learning model that recommends finance actions based on customer order history and financial information.

The goal is to assist finance teams by providing data-driven recommendations, confidence scores, and explainable AI insights.

---

## Features

* XGBoost classification model
* Train/test split
* Cross-validation
* Hyperparameter tuning
* Feature importance analysis
* Confusion matrix
* SHAP explainability
* Prediction confidence scores
* Automated PDF report generation

---

## Example Recommendations

The model can recommend actions such as:

* Release Order
* Send Statement
* Review Open Invoices
* Review Credit Hold
* Require Prepayment
* Escalate to Finance
* Approve Credit Increase
